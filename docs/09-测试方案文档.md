## 2. 测试环境准备

### 2.1 依赖服务启动

#### 后端服务启动
```bash
# 进入项目根目录
cd F:\JavaCode\rag-enterprise-doc-qa

# 使用Maven运行Spring Boot应用
mvn spring-boot:run -pl rag-api

# 服务启动后访问地址
# 后端API: http://localhost:8080/api
# Swagger文档: http://localhost:8080/doc.html
```

#### 前端服务启动
```bash
# 进入前端目录
cd F:\JavaCode\rag-enterprise-doc-qa\rag-front

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev

# 前端访问地址: http://localhost:5173
```

## 6. 测试执行顺序

1. **基础检查**：启动后端服务 → 测试 `/api/health`
2. **文档管理**：上传文档 → 查询列表 → 更新状态 → 删除文档
3. **智能问答**：上传测试文档 → 在/chat页面提问 → 验证回答和来源
4. **历史记录**：完成问答后 → 查看/history页面 → 验证记录保存
5. **系统配置**：访问/settings → 修改参数 → 保存验证
